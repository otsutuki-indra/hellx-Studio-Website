💾 DATABASE SCHEMA

   USERS
   ├── id (PK)
   ├── email (unique)
   ├── username (unique)
   ├── password_hash
   ├── credits (default: 100)
   ├── tier (free/pro/premium)
   └── avatar_url

   CONVERSATIONS
   ├── id (PK)
   ├── user_id (FK) → users.id
   ├── title
   ├── topic
   └── timestamps

   MESSAGES
   ├── id (PK)
   ├── conversation_id (FK) → conversations.id
   ├── role (user/assistant)
   ├── content
   └── tokens_used

   SESSIONS
   ├── id (PK)
   ├── user_id (FK) → users.id
   ├── token (unique)
   └── expires_at

   CREDIT_TRANSACTIONS
   ├── id (PK)
   ├── user_id (FK) → users.id
   ├── amount
   ├── type (purchase/usage/bonus)
   └── description

   INDEXES: 7 (optimized for concurrent access)
